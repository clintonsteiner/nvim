===========
Keybindings
===========

Leader key is ``<Space>``.
Use ``which-key`` by pressing ``<leader>`` and pausing briefly.

Daily Essentials
================

+-------------------+-------------------------------------------+
| Key               | Action                                    |
+===================+===========================================+
| ``<leader>of``    | Open project files                        |
+-------------------+-------------------------------------------+
| ``<leader>ja``    | Add current file to Harpoon               |
+-------------------+-------------------------------------------+
| ``<leader>j1..j4``| Jump to Harpoon slots                     |
+-------------------+-------------------------------------------+
| ``<leader>mn``    | Run nearest test (Go/Python)              |
+-------------------+-------------------------------------------+
| ``<leader>mt``    | Run test task                             |
+-------------------+-------------------------------------------+
| ``<leader>ml``    | Run lint task                             |
+-------------------+-------------------------------------------+
| ``<leader>mb``    | Run build task                            |
+-------------------+-------------------------------------------+
| ``<leader>qo``    | Open quickfix                             |
+-------------------+-------------------------------------------+
| ``<leader>qc``    | Close quickfix                            |
+-------------------+-------------------------------------------+

Git and Rebase
==============

+-------------------+-------------------------------------------+
| Key               | Action                                    |
+===================+===========================================+
| ``<leader>gg``    | Git status picker                         |
+-------------------+-------------------------------------------+
| ``<leader>gRm``   | Rebase onto ``origin/master``             |
+-------------------+-------------------------------------------+
| ``<leader>gRc``   | Rebase continue                           |
+-------------------+-------------------------------------------+
| ``<leader>gRa``   | Rebase abort                              |
+-------------------+-------------------------------------------+
| ``<leader>gRl``   | List unresolved conflicts                 |
+-------------------+-------------------------------------------+
| ``<leader>gU``    | Fetch upstream (fallback origin)          |
+-------------------+-------------------------------------------+
| ``<leader>gF``    | Fetch ``upstream`` for fork workflows     |
+-------------------+-------------------------------------------+
| ``<leader>gPc``   | Run PR checks (lint/test/build)           |
+-------------------+-------------------------------------------+
| ``<leader>gPo``   | Create PR via ``gh``                      |
+-------------------+-------------------------------------------+

Project Setup Commands
======================

+-------------------+-------------------------------------------+
| Command           | Action                                    |
+===================+===========================================+
| ``:ProjectInit``  | Create ``.nvim/tasks.lua`` template       |
+-------------------+-------------------------------------------+
| ``:ProjectOverrideTrust`` | Trust project overrides            |
+-------------------+-------------------------------------------+
| ``:GitFetchForkUpstream`` | Fetch ``--prune upstream``         |
+-------------------+-------------------------------------------+
